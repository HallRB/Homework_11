Versapp, a versatile note taking app.

Description

  • Versapp gives the the user the ability to generate notes they could edit, delete, or view later as it's saved on the server

    • Installation





      • Usage

      • After installing the necessary extensions, open the terminal and enter "node app.js". Open a webpage and go to "localhost:8080". Once the page is open, click the title "Versapp" to enter the notes page. Once you have entered a title and text in the body of a note on the right, the option to save will appear as a save icon in the top right part of the page. Clicking this will save the note allowing you to return to it to make any changes. The saved note will appear on the left side of the page. A pen icon on the top right part of the page will allow you to start a new note.
